Почему нужен Client ID: Это просто идентификатор твоего приложения в Discord, не секретный ключ. Он нужен чтобы Discord знал как называть активность и какие иконки использовать. Создаётся бесплатно за 30 секунд. Как получить: Заходишь на discord.com/developers/applications Нажимаешь New Application и вводишь название, например SoundCloud Копируешь Application ID — это и есть твой Client ID В разделе Rich Presence -> Art Assets загружаешь иконку soundcloud_logo Вставляешь ID сюда
